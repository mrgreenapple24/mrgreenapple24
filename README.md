### 🛠️ What I'm Currently Working On

* **[RISC-V CPU]:** Designing and implementing a RISC-V processor in **Verilog** for FPGA deployment, focusing on custom control units and instruction decoding.
* **[NES Emulator]:** Building a cycle-accurate emulator for the NES, focusing on the 6502 CPU and PPU implementation.
* **[DCT-2 FPGA Implementation]:** Implementing a hardware-accelerated 2D Discrete Cosine Transform (Type II) in Verilog for image compression tasks.
* **[2D Game Engine]:** Developing a high-performance 2D engine in **C++** with custom rendering and physics.
* **[Chess Engine]:** Writing a chess engine from scratch, focusing on move generation, bitboards, and search optimization.
